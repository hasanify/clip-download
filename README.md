# ClipDownload

A simple, premium React/Vite application that allows you to paste images from your clipboard and download them as PNG files instantly.

## Features
- **Screen-Wide Capture:** Just press your paste shortcut anywhere on the screen—no specific zone required.
- **OS-Aware:** Automatically displays `Ctrl+V` or `Cmd+V` based on your operating system.
- **Local Processing:** No images are ever uploaded to a server. Processing happens entirely in your browser.
- **Premium Design:** Immersive glassmorphic UI with smooth transitions and animating gradients.
- **Secure:** Privacy-first architecture.

## Tech Stack
- **Framework:** React / Vite (TypeScript)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Notifications:** Sonner

## Development

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run dev server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
